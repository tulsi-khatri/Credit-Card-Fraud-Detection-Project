# Credit-Card-Fraud-Detection-Project
Implemented a machine learning pipeline for detecting fraudulent credit card transactions.  - Performed data preprocessing and feature scaling using StandardScaler - Handled imbalanced dataset analysis - Trained Logistic Regression model for baseline classification - Applied Isolation Forest for anomaly-based fraud detection - Evaluated both models using accuracy, confusion matrix, and classification report - Compared supervised vs unsupervised approach for fraud detection  This project demonstrates practical ML application on real-world imbalanced dataset.
